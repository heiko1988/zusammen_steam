# Steam Spielvergleich

Vergleiche Steam-Spiele mit Freunden – **100% offline-fähig**, **kein Server**, **GitHub Pages**.

## Setup
1. **Forke dieses Repo**
2. Gehe zu [Steam API Key](https://steamcommunity.com/dev/apikey) → erstelle Key
3. Öffne `assets/script.js` → ersetze `DEIN_STEAM_API_KEY`
4. Ändere `ADMIN_PASSWORD` im JS
5. Aktiviere **GitHub Pages** in den Settings
6. Fertig! → `https://deinname.github.io/steam-compare/`

## Admin
- Gehe zu `admin.html`
- Passwort: (dein gesetztes Passwort)
- Button: "Alle neu laden", "Cache leeren"
